Hertz + Kitex 



## 技术栈

| 功能      | 实现             |
|---------|----------------|
| HTTP 框架 | Hertz          |
| RPC 框架  | Kitex          |
| 数据库     | MySQL、Redis    |
| 身份鉴权    | Paseto         |
| 服务与配置中心 | Consul         |
| 消息队列    | RabbitMQ       |
| 服务治理    | OpenTelemetry  |
| 指标监控    | Prometheus     |
| 链路追踪    | Jaeger         |
| 限流熔断    | Sentinel       |
| 对象存储    | MinIO          |
| 图像识别    | 百度 OCR         |
| CI      | GitHub Actions |
